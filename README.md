# 💖 Eternal Valentine's Magic | 3D Love Proposal 💎

An **Ultra Premium Romantic Interactive Website** built using **HTML, CSS & JavaScript**.
This project creates a **3D Valentine Proposal Experience** with animations, music, and interactive UI.

---

## 🚀 Live Features

✨ 3D Tilt Card (Mouse movement effect)
💞 Rotating Avatar (3D Flip Animation)
🌸 Floating Hearts Background
😂 Smart Moving "NO" Button
🎉 Heart Rain + Confetti Celebration
🎵 Romantic Background Music (Web Audio API)
🎥 Background Video (Cinematic Feel)
💎 Glassmorphism UI Design
📱 Fully Responsive (Mobile Friendly)

## 🎯 How It Works

### ❤️ YES Button

* Triggers:

  * 💖 Heart Rain
  * 🎉 Confetti Blast
  * ✨ Sparkle Effects
  * 🎵 Romantic Music
* Updates message & UI

### 😅 NO Button

* Moves randomly on hover/click
* Displays funny romantic messages
* Makes it impossible to click 😆
